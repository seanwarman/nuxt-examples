# Usage

## Size

To set the size of the icon use the tailwind width classes on the component Elements

<Canvas withSource="open">
  <Story name="size 5">
    {((args) => ({
      components: { TzChevronIcon },
      setup() {
        return { args };
      },
      template: `<tz-chevron-icon class="w-5"/>`
    })).bind()}
  </Story>
  <Story name="size 10">
    {((args) => ({
      components: { TzChevronIcon },
      setup() {
        return { args };
      },
      template: `<tz-chevron-icon class="w-10"/>`
    })).bind()}
  </Story>
</Canvas>

## Colour

To set the colour of the icon set the text colour on the component elements using tailwind classes.

<Canvas withSource="open">
  <Story name="color grey">
    {((args) => ({
      components: { TzChevronIcon },
      setup() {
        return { args };
      },
      template: `<tz-chevron-icon class="w-10 text-gray-400"/>`
    })).bind()}
  </Story>
  <Story name="colour primary">
    {((args) => ({
      components: { TzChevronIcon },
      setup() {
        return { args };
      },
      template: `<tz-chevron-icon class="w-10 text-primary"/>`
    })).bind()}
  </Story>
</Canvas>

## Orientation

To set the orientation of the icon, add a tailwind rotate class to the component element.

<Canvas withSource="open">
  <Story
    name="rotate 0"
    args={{
      class: 'w-10'
    }}
  >
    {((args) => ({
      components: { TzChevronIcon },
      setup() {
        return { args };
      },
      template: `<tz-chevron-icon class="w-10"/>`
    })).bind()}
  </Story>
  <Story name="rotate 90">
    {(args) => ({
      components: { TzChevronIcon },
      setup() {
        return { args };
      },
      template: `<tz-chevron-icon class="w-10 rotate-90"/>`
    })}
  </Story>
  <Story name="rotate 180">
    {(args) => ({
      components: { TzChevronIcon },
      setup() {
        return { args };
      },
      template: `<tz-chevron-icon class="w-10 rotate-180"/>`

</Canvas>
